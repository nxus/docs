Metrics

The Metrics module captures arbitrary data about events (a metric) for storage and querying later.

Instance members

#capture(model, [events])

Captures metric data about a specified @nxus/storage Model

capture(model, [events])

Captures metric data about a specified @nxus/storage Model

Parameters

  • string model :

    The name of the model to capture. Must be a Storage/BaseModel class.

  • Array = (default ['create', 'update', 'destroy']) events :

    Optional. An array of model events to capture. Defaults to 'create', 'update', 'destroy'.
